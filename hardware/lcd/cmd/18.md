# The electronic volume

* To set the screen to pure white, use:
  * libpmdd: `LCD_CONTRAST(0);`
  * C raw:
  	```c
	LCD_CTRL = 0x81;
	LCD_CTRL = 0;
	```
  * ASM:
  	```
	LD LCD_CTRL, #081h
	LD LCD_CTRL, #0
	```
* To set the screen to pure black, use:
  * libpmdd: `LCD_CONTRAST(63);`
  * C raw:
  	```c
	LCD_CTRL = 0x81;
	LCD_CTRL = 63;
	```
  * ASM:
  	```
	LD LCD_CTRL, #081h
	LD LCD_CTRL, #63
	```
* To set the contrast to the default, use:
  * libpmdd: `LCD_CONTRAST(32);`
  * C raw:
  	```c
	LCD_CTRL = 0x81;
	LCD_CTRL = 32;
	```
  * ASM:
  	```
	LD LCD_CTRL, #081h
	LD LCD_CTRL, #32
	```
