# NOP = No Operation

| Hex | Mnemonic | Cycles |
| --- | -------- | ------ |
| FF  | NOP      | 2      |

## Execute

Nothing

## Description

Does nothing, used for short delays.

## Conditions

None

## Examples

```
; Delay 8 Cycles
NOP
```
