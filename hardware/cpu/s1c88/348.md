# S1C88348/317/316/308

Adapted from the [S1C88348/317/316/308 Technical Manual](https://global-origin.epson.com/products_and_drivers/semicon/pdf/id000646.pdf)

## I/O Memory Map

Value meanings for multi-bit values are largely not reproduced in this table.

The following registers are disabled on...

* S1C88308:
  * EBR (R/W)
  * SIK11 (0)
  * KCP11 / K11D (1)
  * HZR51 (R/W)
  * HZR35~HZR37 (R/W)
  * R34D~R37D (R/W)
  * R51D (R/W)
* S1C88317 / S1C88348
  * CEMD1/CEMD0 when in MCU mode (R/W)
