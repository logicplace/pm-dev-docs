# $2001 bit 4

$2001 bit 4 is a [reserved R/W register](../../../glossary.md#reserved-rw-register).

It's set to 0 [after passing the Nintendo check](../../../software/bios/disasm/#user-content-0154) and it's set to 1 during [init](../../../software/bios/disasm/#user-content-00C7) [shutdown](../bios.md#shutdown-system).

This is checked when the power key is pressed and the console is off [here](../../../software/bios/disasm/#user-content-03C6). If it's set, the system restarts instead of waking up.

It's also checked when the cartridge is ejected [here](../../../software/bios/disasm/#user-content-0445). If it's set, the system restarts.

In previous documentation, this register was called "Power on reset" but it's a slightly misleading name. A more correct one might be "reset when powered on".
